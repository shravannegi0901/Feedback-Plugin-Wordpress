This plugin allow you to add youtube video url and select the template from themes and generate shortcode.
Just use the shortcode feedback_embbedids]) anywhere in the code and enjoy beautiful customized feedback.


== Features ==

   User can add comment based on the article if they liked it or not.


== Installation ==

= Install =

1. Login into your WordPress admin panel
2. Navigate to Plugins = Add New
3. Click Upload 
4. Click Choose File and select the Feedback.zip. Click Install Now.
6. Activate the plugin through the 'Plugins' menu in WordPress.
7 Go to dashboard and click on Video Plugins

= Uninstall =

1. Deactivate Video Plugins in the 'Plugins' menu in Wordpress.
2. After Deactivation a 'Delete' link appears below the plugin name, follow the link and confim with 'Yes, Delete these files'.
3. This will delete all the plugin files from the server as well as erasing all options the plugin has stored in the database.
